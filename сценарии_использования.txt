вариант 10 (Клиентская база юридической фирмы)

данное приложение используется только внутри компании по оказанию
юридических услуг и подразумевает наличие всех прав у имеющих доступ к приложению

везде, где в скобках указано "один или несколько",
подразумевается кнопка "+" для добавления дополнительного поля

меню слева
	список услуг
	список должностей
	регистрация договора 
	добавление данных
	
страницы
	главная
	страница добавления данных
	страница списка клиентов
	страница списка сотрудников
	страница списка договоров
	страница списка услуг
	страница регистрации договора
	страница добавления клиента
	страница добавления сотрудника
	страница добавление должности
	страница добавления услуги
	страница профиля клиента
	страница профиля сотрудника
	изменение профиля клиента
	изменение профиля сотрудника
	изменение профиля услуги
	изменение профиля должности
	страница "ошибка"

содержание страниц
	главная
		текст (внутренний сайт для быстрой работы сотрудников с базой)
	
	поиск клиентов
		поле ввода названия фирмы (выпадающий список)
		поле ввода услуг (выпадающий список)
		поле ввода времени оказания услуги (временной интервал)
		поле ввода задействованного сотрудника (набрать фамилию)
	
	результат поиска клиентов
		таблица: название клиента
			 название клиента -- ссылка на профиль клиента
			 сотрудники -- ссылки на их профили
		кнопка перехода назад (поиск клиентов)
	
	поиск сотрудников
		поле ввода фамилии
		поле ввода имени
	
	результат поиска сотрудников
		таблица: ФИО, контакты
			 ФИО -- ссылка на профиль сотрудника
	        кнопка назад -- возврат к поиску сотрудников
		кнопка перехода назад (поиск сотрудников)

	поиск договоров
		поле ввода названия клиента
		поле ввода услуги
		поле ввода интервала времени
	
	результат поиска договоров
		таблица: название услуги, название клиента,
		время оказания услуги, задействованные сотрудники
			 название клиента -- ссылка на профиль клиента
			 сотрудники -- ссылка на их профили
		кнопка перехода назад (поиск договоров)

	список услуг
		название (ссылка на профиль услуги)
		цена
	
	список должностей
	       название (ссылка на профиль должности)
	       зарплата

	регистрация договора
		поле ввода названия услуги (выпадающий список)
		поле ввода названия клиента (выпадающий список)
		поле ввода сотрудника (одного или нескольких) (выпадающий список)
		поле "+" для добавления более одного сотрудника
		поле ввода времени оказания услуги
		кнопка создать
		автоматический переход на страницу "успешно" или на страницу "ошибка"		

	добавление данных
		кнопки перехода на добавление:
		       сотрудника
		       клиента
		       должности
		       услуги

	добавление сотрудника
		3 поля для ввода ФИО
		поле ввода телефона (одного или нескольких)
		поле "+" для добавления более одного телефона
		поле ввода домашнего адреса
		поле ввода почты (один или несколько адресов)
		поле "+" для добавления более одного почтового адреса
		поле ввода должности (выпадающий список)
		поле ввода образования (выпадающий список)
		кнопка создать
		автоматический переход на страницу "успешно" или на страницу "ошибка"
		
	добавление клиента
		поле ввода названия компании
		поле ввода ФИО представителя клиента (одного или нескольких)
			поле ввода телефона представителя (один или
			     несколько номеров)
			поле ввода почты представителя (один или несколько
			     адресов)
			поле ввода рабочего адреса представителя
		кнопка создать
		автоматический переход на страницу "успешно" или на страницу "ошибка"

	добавление должности
		поле ввода названия должности
		поле ввода зарплаты
		кнопка создать
		автоматический переход на страницу "успешно" или на страницу "ошибка"	

	добавление услуги
		поле ввода названия услуги
		поле ввода цены
		кнопка создать
		автоматический переход на страницу "успешно" или на страницу "ошибка"

	профиль услуги
		название
		цена
		кнопка редактировать

	профиль должности
		название
		зарплата
		кнопка редактировать

	профиль клиента
		название компании
		список представителей и их контактов (телефон, почта, рабочий адрес)
		кнопка редактировать данные
		кнопка назад к поиску

	профиль сотрудника
		ФИО
		список телефонов
		список почтовых адресов
		домашний адрес
		должность
		образование
		список договоров с его участием (услуга + название клиента)
		кнопка редактировать данные
		кнопка назад к поиску
	
	изменение профиля клиента/сотрудника/услуги/должности
		поля редактирования всех данных с возможностью
		     добавления новых (кнопка +)
		кнопка удалить профиль (автоматический переход на
		       страницу "успешно")
		кнопка сохранить (автоматический переход на страницу "успешно" или на страницу "ошибка")
		кнопка назад к поиску

		кнопка создать
		автоматический переход на страницу "успешно" или на страницу "ошибка"

	страница "ошибка"
		 текст "ошибка"
		 кнопка назад

	страница "успешно"
		текст "успешно"

Use cases:
    ! поиск клиентов сотрудника (
    в меню слева нажать на поиск, на странице поиска выбрать
    поиск клиентов, выполнится переход на соответствующую страницу,
    вести фамилию сотрудника в поле обслуживающий сотрудник,
    нажать кнопку поиск, получится отфильтрованный список
    клиентов (см. результаты поиска клиентов))
    
    ! поиск контактов сотрудника (в меню слева нажать на поиск, на странице
    поиск выбрать поиск сотрудников, ввести фамилию в поле "фамилия"
    (и имя, опционально, в поле "имя"), нажать кнопку поиск,
    получится список сотрудников (см. результаты поиска сотрудников)
    
    ! поиск контактов клиента (в меню слева поиск - поиск клиентов, ввести
    название клиента в поле "название клиента", нажать поиск,
    получится список клиентов)

    ! поиск договоров клиента (за временной интервал) (в меню слева поиск, нажать 
    на поиск договоров,
    ввести в поле "название клиента" название клиента, ввести временной интервал (опционально)
    нажать поиск, получится список договоров)

    ! поиск договоров за временной интервал (в меню слева поиск,
    нажать на поиск договоров, ввести в поле "временной интервал" интервал,
    нажать кнопку поиск, получится список договоров на услуги,
    оказанные за этот период)

    поиск договоров по услуге за период времени (в меню слева выбрать поиск,
    нажать на поиск договоров, ввести в поле "временной интервал" интервал,
    в поле "услуга" название услуги, нажать кнопку поиск,
    получится список договоров на эту услугу, оказанные за этот период)

    ! поиск всех клиентов по услугам (в меню слева выбрать поиск,
    нажать на поиск клиентов, выбрать в поле "услуга" услугу,
    нажать поиск, получится список клиентов, заказывающих эти услуги)
    
    изменение данных о сотруднике (в меню слева выбрать поиск,
    нажать на поиск сотрудников, ввести в поле "фамилия" фамилию сотрудника,
    в поле "имя" имя (можно ввести только фамилию или только имя)
    нажать кнопку поиск, получится список сотрудников, нажать на ФИО,
    чтобы перейти в профиль, на странице профиля редактировать данные,
    нажать "сохранить")
	      
    регистрация нового договора (в меню слева выбрать регистрация договора,
    выбрать название услуги в поле "услуга", выбрать клиента в поле "клиент",
    выбрать сотрудника в поле "сотрудник", если сотрудников несколько,
    нажать на "+" и добавить ещё сотрудников, ввести время оказания
    услуги в поле "время регистрации", нажать кнопку создать)

    добавление сотрудника (в меню слева выбрать добавление данных,
    нажать на добавление сотрудника, ввести ФИО в соответствующие поля,
    ввести номер телефона в поле "телефон", если номеров несколько,
    нажать + и ввести ещё номер, то же самое для почты, выбрать должность,
    выбрать образование, нажать кнопку создать)

    добавление клиента (в меню слева выбрать добавление данных,
    нажать на добавление клиента, ввести название в поле "название",
    ввести ФИО представителя клиента в соответствующие поля,
    его телефон (если телефонов несколько, нажать + и ввести ещё) и
    почту (если несколько почтовых адресов, нажать + и ввести ещё адрес),
    ввести рабочий адрес представителя, нажать кнопку создать)

    добавление услуги (в меню слева выбрать добавление данных,
    нажать на добавление услуги, ввести название в поле "название",
    ввести цену в поле "цена", нажать кнопку создать)

    добавление должности (в меню слева выбрать добавление данных,
    нажать на добавление должности, ввести название в поле "название",
    ввести зарплату в поле "зарплата", нажать кнопку создать)
	       
    удаление сотрудника (в меню слева выбрать поиск, выбрать поиск
    сотрудников, ввести фамилию в поле "фамилия", опционально -- имя
    в поле "имя", нажать поиск, получится список сотрудников,
    нажать на ФИО, чтобы перейти в профиль, на странице профиля нажать
    кнопку редактировать, на странице редактирования нажать кнопку "удалить
    профиль")

    удаление должности (в меню слева выбрать список должностей,
    на странице списка должностей нажать на название должности,
    чтобы перейти в профиль, на странице профиля нажать
    кнопку редактировать, на странице редактирования нажать кнопку "удалить
    профиль")

    удаление услуги (в меню слева выбрать список услуг,
    на странице списка услуг нажать на название услуги,
    чтобы перейти в профиль, на странице профиля нажать
    кнопку редактировать, на странице редактирования нажать кнопку "удалить
    профиль")

    изменить название/цену услуги (в меню слева выбрать список услуг,
    на странице списка услуг нажать на название услуги,
    чтобы перейти в профиль, на странице профиля нажать
    кнопку редактировать, на странице редактирования изменить название/цену,
    нажать кнопку сохранить)

    изменить зарплату должности (в меню слева выбрать список должностей,
    на странице списка должностей нажать на название должности,
    чтобы перейти в профиль, на странице профиля нажать
    кнопку редактировать, на странице редактирования изменить зарплату,
    нажать кнопку сохранить)
